memcachedb-win32 by mplatov

win32 builld for memcachedb

MemcacheDB is a distributed key-value storage system designed for persistent

Dependencies

pthreads-win32, libevent, berkleyDB

Install

1. install BerkleyDB 4.7.25 or later
2. install pthreads-win32
3. download and compile libevent for win32 (git://github.com/mplatov/libevent.git)
4. build memcachedb
